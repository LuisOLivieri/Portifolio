MY GRID TEC

Tecnologias que uso atualmente

HTML

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets e é usado para estilizar o site.

JAVASCRIPIT

JavaScript é uma linguagem de programação que permite você criar e dar usabilidade ao seu site.

SASS

O SASS é uma linguagem de extensão do CSS, a sigla significa “Syntactically Awesome Style Sheets” traduzindo ao pé da letra, folhas de estilo com uma sintaxe incrível.

jQuery - Estudando

jQuery é uma biblioteca livre que contém funções da linguagem de programação JavaScript que interage com páginas em HTML, desenvolvida para simplificar os scripts executados/interpretados no navegador de internet do usuário.

Bootstrap - Estudando

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.
